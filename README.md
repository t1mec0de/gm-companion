# GM Companion

GM Companion is a Flutter application for tabletop RPG Game Masters.

## Features

- Combat Tracker
- Initiative Management
- Status Effects
- Dice Roller
- Encounter Management (WIP)
- Adaptive Music System (Planned)
- LAN Sessions (Planned)
